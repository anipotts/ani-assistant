# API Directory

This directory contains all API routes using Next.js App Router.

The API follows a versioned structure:

- `/api/v1/` - Base API version 1
- Future versions can be added as `/api/v2/`, etc.

## API Organization

- `auth/` - Authentication endpoints
- `threads/` - Message thread management
- `user/` - User profile and preferences
- `webhooks/` - External service webhooks (Gmail, Twilio)
