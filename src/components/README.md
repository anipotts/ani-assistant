# Components Directory

This directory contains all React components organized into subdirectories:

- `auth/` - Authentication components (login, signup, user dropdown)
- `inbox/` - Thread list and message preview components
- `layout/` - App shell, navigation, and common layout components
- `settings/` - User preferences and settings forms
- `thread/` - Message detail, composer, and delegation components
- `ui/` - Reusable UI components using shadcn/ui
